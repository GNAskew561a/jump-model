Alexander Jump Model

Browser-only interactive model for Squat vs Catapult vs Countermovement jumps.

Controls

Tip: change one parameter at a time and compare the solid line with the faint dotted human-reference line. This version follows Alexander (1995) closely for the CE–SEC mechanics, but includes a finite muscle activation time constant so force rises more realistically.

Geometry and mass

These sliders change body size, starting position, and how much mass is in the moving leg segments.

Muscle properties

These sliders control how much force the muscle can produce, how fast it can shorten, how quickly it activates, and how leverage changes with knee angle.

Tendon property

This slider changes how stretchy the series elastic component is.

Countermovement jump timing

These controls affect when the muscle turns on during the countermovement jump. The tendon mechanics follow Alexander; activation rises with a finite time constant.

Keeps CMJ depth comparable with the chosen starting knee angle.
